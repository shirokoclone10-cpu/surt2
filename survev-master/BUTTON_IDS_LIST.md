## Usage Examples

### Accessing Medical Item Button
```javascript

const sodaButton = document.getElementById('ui-loot-soda');
const painkiller = document.getElementById('ui-loot-painkiller');
```

### Accessing Ammo Display
```javascript
const ammo9mm = document.getElementById('ui-loot-9mm');
const ammo556mm = document.getElementById('ui-loot-556mm');

```

### Accessing Weapon Slots

const healthBar = document.getElementById('ui-health-actual');
